# Handbook Index

Use this folder for handbook sub-pages referenced by `docs/HANDBOOK.md`.

- `projects/` – per-project deep dives (backend, admin dashboard, mobile).
- `generated/` – auto-generated inventories (modules, services, pages, mobile structure). Refresh via `node scripts/atctl.mjs docs --sync`.
- Future sub-folders can capture process docs, ownership matrices, etc.

All files should include **Last Updated** metadata and link back to the handbook.
